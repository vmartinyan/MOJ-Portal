Information about code

Framework

The portal is developed on CMS of Wordpress. The last updated version is - Wordpress 4.9.5

Database

The database is MySQL - Version: 5.5.52

The database backups are stored on /wp-content/aiowps_backups

Updating

For updating (especially version of wordpress) need to fill the credentials of FTP in wp-config.php


define('FTP_PUBKEY','');
define('FTP_PRIKEY','');
define('FTP_USER','');
define('FTP_PASS','');
define('FTP_HOST','127.0.0.1:22');

Here must fill the credentials of FTP user that has appropriate permission for file system
